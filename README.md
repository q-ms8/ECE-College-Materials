# ECE College Materials

Study materials for Electronic & Communication Engineering (ECE) organized by academic year.

## Structure

### Year 2
| Subject | Description |
|---------|-------------|
| **Logic-Design** | Digital logic fundamentals, circuits, and sources |
| **MatLab** | MATLAB labs, tutorials, and assignments |
| **Microelectronics** | Microelectronics course materials |
| **Network-Analysis** | Chapters, exams, homework, notes, and solutions (Ch. 1-19, A-B) |
| **Probability** | Final/midterm exams, MATLAB assignments, slides, sources |
| **Projects** | Lectures, midterms, paper examples and templates |
| **Signals-and-Systems** | Lectures and sources |

### Year 3 - Course I
| Subject | Description |
|---------|-------------|
| **Analog-Electronics-LAB** | Data sheets, exams, lectures, solutions |
| **Communication-Systems-I** | Exams, lectures, lab sheets, sources |
| **English** | English language sources |
| **IC-and-Application** | Integrated circuits lectures and sources |
| **Sequential-Logic-Circuits** | Data sheets, exams, experiments, simulations, tutorials |
| **System-Engineering** | System engineering sources |
| **Wave-Propagation** | Exams, lectures, notes, previous year materials |

### Year 3 - Course II
| Subject | Description |
|---------|-------------|
| **Antenna-Engineering** | Lectures, labs, notes, graph papers, sources |
| **Communication-Design-LAB** | Lectures (HW, LAB, Theory), project, sources |
| **Communication-Systems-II** | Lab sheets, lectures, sources, tutorials |
| **Digital-Design** | FBGA/VHDL projects, lab sheets, lectures, old exams |
| **Electronic-Design-LAB** | Lectures (HWS, LAB, Theory), sources |
| **Power-Electronics** | Lectures (LAB, Theory), notes, sheets, sources |
